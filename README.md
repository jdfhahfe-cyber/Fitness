# Aura Fitness - Professional Edition

A high-performance, AI-powered fitness tracking application built with React, Vite, and Firebase.

## Features

- **Performance Dashboard**: Real-time stats with professional-grade data visualization.
- **AI Training Strategies**: Personalized workout plans generated by Gemini AI.
- **Activity Intelligence**: Log varied sports with intensity-based metrics.
- **Stalorine Integration**: Export your entire performance history into a single, standalone HTML document.
- **Achievement Hall**: Track milestones and unlock badges as you progress.
- **Cloud Sync**: Securely persisted data via Firebase Firestore.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [Firebase account](https://firebase.google.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd aura-fitness
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. Configure Firebase:
   Ensure your `firebase-applet-config.json` is updated with your production Firebase credentials.

5. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment

This app is optimized for deployment on **Vercel**, **Netlify**, or **Firebase Hosting**.

1. Build the production assets:
   ```bash
   npm run build
   ```
2. Deploy the `dist` folder to your provider of choice.
